Docker Container IP

查看容器IP
// 查看具体某一个容器的IP，需要已知容器ID
docker inspect --format '{{ .NetworkSettings.IPAddress }}' <container-ID>
// 查看所有容器的IP
docker inspect --format='{{.Name}} - {{range .NetworkSettings.Networks}}{{.IPAddress}}{{end}}' $(docker ps -aq)

默认docker0的段是：172.17.0.1/16
